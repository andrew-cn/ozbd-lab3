## Мета роботи

Набуття практичних навичок роботи з SQL-запитами на базі локальної бази даних:

- Операції маніпулювання даними (DML): INSERT, UPDATE, DELETE
- Простий вибірковий аналіз (DQL): SELECT з WHERE, AND, OR, IN, LIKE, ORDER BY
- Аналітичні та комплексні запити: агрегатні функції, JOIN, підзапити, UNION
